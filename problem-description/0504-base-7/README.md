## [504. Base 7](https://leetcode.com/problems/base-7)

<code>Easy</code> <code>Math</code>

<br> 

Given an integer <code>num</code>, return *a string of its __base 7__ representation*.

<br> 

#### Example 1:

> __Input:__ num = 100
> 
> __Output:__ "202"  

#### Example 2:

> __Input:__ num = -7
> 
> __Output:__ "-10"  
 
<br> 

#### Constraints:

<code>-10<sup>7</sup> <= num <= 10<sup>7</sup></code>
